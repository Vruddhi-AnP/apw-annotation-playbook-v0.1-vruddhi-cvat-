Annotator Certification Gate Checklist

This checklist confirms that an annotator is ready to start execution work.

---

Inputs Ready:
- [ ] Annotator has access to the annotation tool
- [ ] Label specification document is shared
- [ ] Edge case library is shared

Stop-the-line:
- [ ] If annotator has not read the label spec, stop

---

Process OK:
- [ ] Annotator has completed training quiz
- [ ] Quiz pass criteria is met
- [ ] Sample annotation is reviewed once

Stop-the-line:
- [ ] If quiz is failed, do not assign batches

---

Outputs OK:
- [ ] Annotator is marked as certified
- [ ] Annotator is approved for batch assignment

Checklist Status:
- [ ] PASS – Annotator can start execution
- [ ] FAIL – Training or clarification needed
